We have a lot of features on the DAKU platform. Here below, you can see a description of the primary features along with a video tutorial on how to use the respective feature on the platform.


1. Login/Register

For using the app features, a user has to register with our DAKU app with a valid email id, and password.

Once a user has registered with DAKU, they have to login into their account to access the features of our app. Login can be done using the email address and password used while registering with the app.

2. File Upload and Download

For uploading a file into the file system, the user has to first select the upload file option from the Functions drop down menu. The user can select the file that they want to upload and click on the upload file button. Our app also shows the status of the upload.

My files option shows all the files corresponding to a user account in a list format. A user can save any kind of file in our filesystem like image file, txt file, code files, pdfs etc. There is no restriction on the type of file which can be stored in our file system.

For the list of files, a user can download any file that they want using the download button.

3. Delete File, Restore and Permanent Delete

A user can also delete any file they want. The deleted files are no longer shown in the My Files section of the app.

The deleted files are stored inside the bin folder of our app. On opening the recycle bin, it shows the list of all the files that the user has deleted. On the deleted file, a user can perform two operations:

1. Restore File

2. Permanently Delete the File
